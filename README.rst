Installation
------------

**Latest official release**

.. code-block:: bash
   
   $ pip install pyjob

**Source code**

.. code-block:: bash
   
   $ git clone https://github.com/fsimkovic/pyjob.git
   $ cd pyjob
   $ python setup.py install
